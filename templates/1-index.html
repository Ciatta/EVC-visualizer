Here is a tool for visualizing Eternal Vertex Cover over grid graphs.

Step 1: Create the grid chosing height, width and the grid shape (please avoid grids bigger than 20x20)
Step 2: Select the edge you want to attack (eg. (1,2)) and watch how guards move in order to protect the edge and keep all vertex covered

What is an Eternal Vertex Cover?

An eternal vertex cover is a set of vertices in a graph such that, as edges are attacked over time, it is always possible to move guards within the graph in response to these attacks such that every edge is covered by at least one guard at all times.
